# Week1 - 스프링 입문 (김영한) 완강

## 학습 내용
김영한님의 스프링 입문 강의를 완강하며 스프링의 기본 개념과 백엔드 개발 흐름을 익힘.

### 학습한 주제
- 스프링 프로젝트 환경 설정 및 Gradle 기반 프로젝트 구성
- 스프링 웹 개발 방식 이해
    - 정적 컨텐츠
    - MVC 패턴
    - API 방식
- 스프링 빈과 의존관계(DI) 이해
- 회원 관리 예제를 통한 계층 구조 학습
    - Controller
    - Service
    - Repository
- 다양한 데이터 접근 기술 경험
    - 순수 JDBC
    - JdbcTemplate
    - JPA
    - 스프링 데이터 JPA
- AOP 개념 및 활용 기초 학습


## 다음 목표
- 스프링 핵심 원리 강의로 DI, IoC, 빈 생명주기 더 깊게 학습
- JPA 활용과 스프링 백엔드 프로젝트 실습 진행
